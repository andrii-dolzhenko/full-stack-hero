# React Vite Template

Starter template for React homework projects powered by Vite.

## Files

- `package.json` - project scripts and dependencies.
- `vite.config.js` - Vite configuration.
- `index.html` - Vite HTML entry file.
- `src/main.jsx` - React entry point.
- `src/App.jsx` - starter component.
- `src/App.css` - component styles.
- `public/` - static public assets.

## Usage

Copy this folder, rename it for a new React homework assignment, then run:

```bash
npm install
npm run dev
```

The `package-lock.json` file will be generated inside the copied project after `npm install`.
