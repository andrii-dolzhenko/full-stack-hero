Експеримент 7

Позиціонування елементів

На цій сторінці показано різні значення властивості position: static, relative, absolute, fixed і sticky.

Що робить position

Властивість position визначає, як елемент розташовується на сторінці та чи можна зміщувати його за допомогою властивостей top, right, bottom і left.

За замовчуванням елементи мають position: static. Інші значення дозволяють зміщувати елемент або прив’язувати його до батьківського контейнера, вікна браузера чи моменту прокрутки.

static і relative

static — стандартне положення елемента в потоці документа. relative залишає місце елемента в потоці, але дозволяє візуально змістити його.

position: static

Перший блок
static
Третій блок

Static-елемент розташовується звичайно і не реагує на top або left.

position: relative

Перший блок
relative
Третій блок

Relative-елемент зміщується візуально, але його початкове місце в потоці документа зберігається.

absolute

position: absolute виймає елемент зі звичайного потоку документа. Якщо батьківський блок має позиціонування, absolute орієнтується саме на нього.

Візуальна область контейнера
absolute

Помаранчевий блок розташований відносно пунктирної області, тому що саме вона має position: relative. Синій блок у цьому прикладі лише візуально показує межі батьківського контейнера.

fixed і sticky

fixed прив’язує елемент до вікна браузера, а sticky поводиться як звичайний елемент, доки під час прокрутки не досягне заданої позиції.

position: fixed

fixed

Fixed-елемент прив’язується до вікна браузера. У реальному інтерфейсі так часто роблять кнопки допомоги, чати або плаваючі панелі.

На цій сторінці fixed-блок показаний як безпечна демонстрація всередині макета, щоб він не перекривав меню, текст і кнопки.

position: sticky

sticky

Sticky-елемент спочатку знаходиться у звичайному потоці, але під час прокрутки може “прилипати” до заданої позиції.

Це зручно для заголовків секцій, бокових меню або підказок, які мають залишатися видимими певний час.

Sticky відрізняється від fixed тим, що не завжди прикріплений до екрана, а активується тільки під час прокрутки.

Приклад CSS-коду

Нижче показано кілька базових прикладів позиціонування елементів. 

.relative-box {
  position: relative;
  top: 16px;
  left: 20px;
}

.absolute-box {
  position: absolute;
  top: 24px;
  right: 24px;
}

.sticky-box {
  position: sticky;
  top: 20px;
}

Мій висновок

Властивість position допомагає керувати розташуванням елементів. static залишає елемент у звичайному потоці, relative дозволяє змістити його без втрати місця, absolute розташовує елемент відносно позиціонованого контейнера, fixed прив’язує його до вікна браузера, а sticky поєднує звичайну поведінку з “прилипанням” під час прокрутки.
