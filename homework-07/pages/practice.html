Додаткова практика

Конструктор із блоків

На цій сторінці зібрано просту композицію з блоків. Вона допомагає закріпити роботу з розмірами, кольорами, рамками та позиціонуванням елементів.

Практичне завдання

У цьому прикладі кілька блоків розміщені всередині одного батьківського контейнера. Для побудови використано position: relative у контейнера та position: absolute у внутрішніх блоків.

Батьківський блок створює робочу область, а дочірні блоки отримують точні розміри, кольори та позиції. Так можна зібрати простий “конструктор” і побачити, як елементи накладаються або стають поруч.

Які властивості використано

Для створення композиції використано властивості width, height, background, border, position, top, left, z-index і display. 

.constructor {
  position: relative;
  width: 920px;
  height: 440px;
}

.constructor-left {
  position: absolute;
  top: 40px;
  left: 50px;
  width: 270px;
  height: 270px;
  border: 3px solid #16213e;
}

.constructor-gray {
  position: absolute;
  top: 40px;
  left: 345px;
  width: 540px;
  height: 270px;
}

.constructor-red {
  top: 28px;
  left: 27px;
  width: 480px;
  height: 205px;
}

.constructor-blue {
  top: 230px;
  width: 181px;
  height: 180px;
}

Мій висновок

Ця практика показує, що блокову модель можна використовувати не тільки для звичайного розміщення тексту, а й для побудови візуальних композицій. Кожен блок має власні розміри, фон, рамку і координати, а результат залежить від того, як ці значення поєднуються між собою.
