# Homework 07: CSS Box Model Experiments

> A multi-page educational project that demonstrates how HTML elements behave under different CSS layout rules.

This project was created as part of a full-stack development course. It focuses on the **CSS box model**, element display behavior, browser style normalization, resets, floating elements, overflow handling, positioning, and practical block-based layout construction.

## Project Purpose

The main goal of this project is to show how CSS properties affect the size, position, flow, and visual behavior of HTML elements. Each page explains and demonstrates a separate layout concept through examples, styled blocks, comparison sections, and short code snippets.

## What The Project Does

- Explains the difference between **block** and **inline** elements.
- Demonstrates how `normalize.css` and `reset.css` affect default browser styles.
- Shows how `display` values change element behavior.
- Compares `content-box` and `border-box` sizing models.
- Demonstrates `float` and `clear` behavior.
- Shows how `overflow` controls content that does not fit inside a container.
- Explains CSS positioning with `static`, `relative`, `absolute`, `fixed`, and `sticky`.
- Includes a practical block constructor built with positioned elements.

## Pages Included

| Page | Description |

| `index.html` | Main page with project navigation and a short overview of all sections. |

| `reset.html` | Demonstrates a full CSS reset and its effect on page elements. |
| `display-box.html` | Explains `display`, `box-sizing`, and element size calculation. |

| `positioning.html` | Compares the main CSS positioning types. |
| `practice.html` | Contains a practical visual block constructor using CSS positioning. |
